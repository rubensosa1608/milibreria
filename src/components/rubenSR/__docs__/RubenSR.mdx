# RubenSR

Componente personalizado que muestra un testimonio con autor y avatar.

#### Ejemplo

<Canvas of={RubenSRStories.Default} />

## Uso

```ts
import { RubenSR } from 'milibreria';

function Example(){
  return (
    <RubenSR
      description="Este es un testimonio increíble. ¡Altamente recomendado!"
      author="John Doe"
      avatarUrl="https://via.placeholder.com/150"
      width={150}
      height={150}
    />
  );
}

export default Example;
